# Ecommerce Store

Một ứng dụng bán hàng trực tuyến được xây dựng với React.js và Node.js

## 🛠️ Công nghệ sử dụng

### Frontend
- React.js
- Ant Design
- Vite
- React Router DOM

### Backend  
- Node.js
- Express.js
- PostgreSQL
- JWT Authentication

## 📦 Cài đặt

### Backend
```bash
cd backend
npm install
npm run dev

cd frontend
npm install  
npm run dev
